PHMBench: A comprehensive evaluation suite for prognostic health management algorithms with standardized datasets, metrics, and evaluation protocols.

Abstract

We present PHMBench, a comprehensive benchmark suite designed to standardize the evaluation of prognostic health management algorithms. Our benchmark includes 15+ carefully curated datasets, standardized evaluation metrics, and automated assessment tools that enable fair comparison of PHM methods across different domains. PHMBench covers critical PHM tasks including remaining useful life (RUL) prediction, fault diagnosis, anomaly detection, and health assessment. With over 50 baseline implementations and a continuously updated leaderboard, PHMBench serves as the de facto standard for PHM algorithm evaluation, facilitating reproducible research and accelerating progress in the field.

Evaluation Protocol

Submission Guidelines

  1. Code Submission: Submit your implementation via GitHub repository with clear documentation
  2. Reproducibility: Include environment setup, dependencies, and exact hyperparameters
  3. Evaluation: Your model will be evaluated on held-out test sets using our automated system
  4. Fair Comparison: All methods use identical train/validation/test splits and evaluation metrics

Standard Evaluation Metrics

Task Primary Metric Secondary Metrics Description
RUL Prediction RMSE MAE, Score, R² Root mean square error with asymmetric scoring
Fault Classification F1-Score Accuracy, Precision, Recall Weighted F1-score for multi-class problems
Anomaly Detection AUC-ROC Precision@K, AUC-PR Area under ROC curve for detection performance
Health Assessment Correlation Monotonicity, Trendability Correlation with ground truth degradation

Quick Start

# Install evaluation toolkit
pip install phm-bench

# Download datasets
phm-bench download --dataset nasa-cmapss

# Run evaluation
phm-bench evaluate --method your_method.py --dataset nasa-cmapss --task rul

# Submit results
phm-bench submit --results results.json --method-name "YourMethod"
